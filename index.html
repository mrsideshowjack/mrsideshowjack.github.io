About me

Hi, I’m Jack and I am aspiring web developer currently studying Web Technologies at The University of Portsmouth; I really enjoy finding a new exciting web technologies and experimenting and playing with them. Such as web components, which allows exciting new ways to code for the web, such as custom elements and HTML imports. You can see some experiments I’ve done using Polymer on my Github. Recently I have been shifting my focus to not just coding but also graphic and logo design. I’ve discovered how rewarding seeing your design being used in a final product.

Here are some of the technologies I am experienced with:

  • HTML 5, CSS3 & JS

  • jQuery & Other JS libraries, such as Polymer

  • Git & Github

  • Adobe Photoshop, Illustrator & After Effects

  • node, npm, Bower & Gulp

  • MongoDB & Firebase NoSQL databases.

  • PHP & MySQL databases.
